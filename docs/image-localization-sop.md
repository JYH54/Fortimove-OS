# 이미지 현지화 및 재가공 가이드 (Image Localization SOP)

**목적**: 타오바오/티몰 상세페이지 이미지를 한국 이커머스 플랫폼(네이버 스마트스토어, 쿠팡)에 최적화된 형태로 재가공하여, 초상권·유아 이미지 리스크를 방지하고 판매 전환율을 극대화합니다.

## 1. 적용 범위

*   **Fortimove Global**: 테스트 상품 및 반복 판매 상품의 빠른 이미지 현지화
*   **Fortimove**: 프리미엄 브랜드 이미지 일관성 유지를 위한 고품질 재가공

## 2. 이미지 재가공 필수 처리 항목

### 2-1. 텍스트 현지화

모든 중국어/영문 텍스트를 한국어 판매형 문장으로 변환합니다.

*   **OCR 추출**: Tesseract, Google Cloud Vision 등을 활용하여 이미지 내 텍스트와 위치 좌표 추출
*   **번역 원칙**:
    *   과장 광고 배제 ("세계 최고" → "고품질")
    *   의료기기 오인 표방 차단 ("혈압 강하" → "일상 편안함")
    *   자연스러운 한국어 어순 및 띄어쓰기
*   **폰트 사용 제한**: 저작권 없는 무료 폰트만 사용 (나눔고딕, 프리텐다드, Noto Sans KR)

### 2-2. 리스크 이미지 탐지 및 처리

| 리스크 유형 | 탐지 방법 | 처리 방안 | 우선순위 |
|-----------|---------|---------|---------|
| **유아/아기 이미지** | CLIP 모델 유사도 분석 | 즉시 삭제 또는 성인 이미지로 대체 | **P0** |
| **인물 초상권** | Face Detection (얼굴 인식) | AI 생성 인물로 교체 또는 배경만 유지 | **P1** |
| **타사 브랜드 로고** | OCR + 브랜드 DB 매칭 | 로고 영역 제거 또는 흰색 덮기 | **P1** |
| **의료 효능 문구** | 키워드 필터링 | 중립적 표현으로 변환 | **P0** |

**처리 예시**:
```
[원본 이미지]
- 유아가 상품을 사용하는 장면
- 중국어 텍스트: "宝宝专用 100%安全"

[처리 후]
- 유아 이미지 삭제 → 상품 단독 촬영 이미지로 대체
- 한국어 텍스트: "안심 소재 데일리 케어"
```

### 2-3. 무드톤 조정

사용자가 지정한 브랜드 컨셉에 맞춰 색감을 재조정합니다.

| 무드톤 | 색상 조정 | 대상 브랜드 |
|-------|---------|-----------|
| **프리미엄** | 채도 -30%, 명도 +20%, 화이트/베이지 강조 | Fortimove (메인 브랜드) |
| **가성비** | 채도 +30%, 대비 강조, 비비드 컬러 | Fortimove Global (테스트 상품) |
| **미니멀** | 그레이스케일 또는 단색 배경, 여백 증가 | Fortimove (PB 후보) |
| **트렌디** | 그라데이션, 파스텔톤, SNS 감성 | Fortimove Global (반복 판매) |

### 2-4. 레이아웃 최적화

네이버/쿠팡 상세페이지 가이드라인에 맞춰 재구성합니다.

*   **권장 해상도**: 가로 800px~1200px (모바일 최적화)
*   **이미지 순서**: 메인 비주얼 → 제품 특징 → 스펙 → 사용 시나리오 → 구매 혜택
*   **텍스트 가독성**: 최소 폰트 크기 24px, 명조체 지양 (고딕체 사용)

## 3. 처리 워크플로우 (7단계)

```
[1단계] 이미지 업로드 및 전처리
  → 파일명 자동 정리 (product_detail_01.jpg)
  → 해상도 확인 (800px 미만 시 경고)

[2단계] OCR 텍스트 추출
  → Tesseract (중국어: chi_sim, chi_tra)
  → 텍스트 좌표 및 신뢰도 기록

[3단계] AI 번역 (Claude/GPT-4)
  → 이커머스 최적화 프롬프트 적용
  → 금지 키워드 자동 필터링

[4단계] 리스크 이미지 탐지
  → 유아 탐지 (CLIP 유사도 > 0.7)
  → 인물 탐지 (Face Detection)
  → 브랜드 로고 탐지 (OCR + DB 매칭)

[5단계] 이미지 대체/삭제
  → 유아: 삭제 또는 상품 단독 이미지로 교체
  → 인물: Stable Diffusion 생성 인물로 교체
  → 로고: 영역 제거 (inpainting)

[6단계] 무드톤 조정 및 텍스트 오버레이
  → OpenCV 색상 변환
  → PIL을 이용한 한국어 텍스트 삽입

[7단계] 최종 출력 및 메타데이터 생성
  → JPG/PNG 파일 저장
  → SEO 메타데이터 (상품명, 태그) 자동 생성
```

## 4. SEO 메타데이터 자동 생성

재가공된 이미지와 함께 다음 메타데이터를 자동 생성합니다.

### 4-1. SEO 상품명 (3안 제공)

**구조**: `[카테고리 키워드] + [핵심 USP] + [세부 스펙]`

**예시**:
```
원본 (중국어): 超强吸水速干毛巾 30*70cm 蓝色

생성 결과:
- 안1: 초강력 흡수 속건 마이크로화이버 타월 중형 블루
- 안2: 프리미엄 극세사 타월 빠른건조 30x70 스포츠용
- 안3: 호텔급 속건 타월 흡수력 3배 헬스 헤어타올 블루
```

### 4-2. 검색 태그 (10개)

연관 키워드 및 롱테일 키워드 조합

**예시**:
```
속건타월, 마이크로화이버, 헬스타올, 스포츠수건, 빠른건조,
대용량타월, 흡수력좋은수건, 극세사타올, 여행용수건, 프리미엄타월
```

### 4-3. 옵션명 정규화

중국어/영문 옵션을 직관적 한글로 변환

| 원본 | 정규화 | 비고 |
|-----|-------|------|
| 蓝色 / Blue | 블루 | 색상 |
| 30*70cm | 중형 (30x70cm) | 사이즈 |
| 超大号 | 라지 (대형) | 사이즈 |
| 3条装 | 3장 세트 | 수량 |

### 4-4. 핵심 키워드 (5개)

쿠팡 검색광고용 단일 키워드

**예시**:
```
속건타월, 마이크로화이버타월, 헬스타올, 흡수력타월, 극세사수건
```

## 5. 금지 사항 및 제재 위험 통제

### 절대 금지 항목

| 금지 항목 | 이유 | 처리 방안 |
|---------|-----|---------|
| **유아 이미지 유지** | 플랫폼 어린이제품법 제재 | 무조건 삭제 또는 성인 이미지로 대체 |
| **타사 로고 노출** | 지식재산권 침해 | 로고 영역 자동 제거 |
| **의료 효능 문구** | 의료기기법 위반 | "치료" → "케어", "완치" → "관리" |
| **유료 폰트 사용** | 저작권 침해 | 무료 폰트만 사용 (나눔고딕, 프리텐다드) |
| **과도한 보정** | 허위/과장 광고 | 원본 색감 대비 ±30% 이내 조정 |

## 6. 출력 파일 관리 규칙

### 파일명 규칙
```
{상품가칭}_{이미지순서}_{무드톤}_kr.jpg

예시:
- 속건타월_01_premium_kr.jpg
- 속건타월_02_premium_kr.jpg
```

### 저장 위치
```
클라우드 스토리지 (Google Drive / AWS S3)
└── Fortimove-OS/
    └── products/
        └── {상품가칭}/
            ├── original/  (원본 이미지)
            ├── processed/ (재가공 이미지)
            └── metadata/  (SEO 데이터 JSON)
```

### 메타데이터 JSON 예시
```json
{
  "product_id": "taobao_998630245936",
  "processed_date": "2026-03-27",
  "moodtone": "premium",
  "risk_detected": {
    "infant_images": 2,
    "person_faces": 3,
    "brand_logos": 1
  },
  "risk_processed": {
    "infant_removed": 2,
    "faces_replaced": 3,
    "logos_removed": 1
  },
  "seo": {
    "product_names": [
      "초강력 흡수 속건 마이크로화이버 타월 중형 블루",
      "프리미엄 극세사 타월 빠른건조 30x70 스포츠용"
    ],
    "tags": ["속건타월", "마이크로화이버", "헬스타올"],
    "keywords": ["속건타월", "흡수력타월"]
  }
}
```

## 7. 실무 체크리스트

| 단계 | 확인 항목 | 완료 (O/X) |
|-----|---------|-----------|
| **이미지 전처리** | 해상도 800px 이상 확인 | |
| **OCR 추출** | 중국어 텍스트 추출 완료 | |
| **번역** | 금지 키워드 필터링 완료 | |
| **리스크 탐지** | 유아/인물/로고 탐지 완료 | |
| **리스크 처리** | 유아 이미지 0개 확인 | |
| **무드톤** | 지정 무드톤 적용 완료 | |
| **텍스트 삽입** | 한국어 텍스트 가독성 확인 | |
| **SEO 생성** | 상품명 3안 + 태그 10개 생성 | |
| **파일 저장** | 파일명 규칙 준수 확인 | |
| **메타데이터** | JSON 파일 저장 완료 | |
