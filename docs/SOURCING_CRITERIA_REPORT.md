## 📌 Executive Summary

Fortimove는 **3단계 소싱 검증 체계**를 통해 법적 리스크를 최소화하고 고마진 상품을 선별합니다.

### 핵심 소싱 철학

1. **안전 제일**: 지재권/통관/의료기기 리스크 0% 목표
2. **고마진 전략**: 마진율 45% 이상 자동 승인 (Golden Pass)
3. **큐레이션 브랜드**: 단순 직구 대행이 아닌 전문가 검수 서비스
4. **규제 준수**: 2026년 강화된 관세법/KC인증/식약처 규정 완벽 대응

## 2️⃣ 1단계: Rule-based 리스크 필터링

### 2.1 리스크 카테고리별 감지 키워드

Fortimove는 **9개 리스크 카테고리**에 대해 자동 감지합니다.

#### 🚨 치명적 리스크 (즉시 차단)

##### 1. **지재권 리스크**
**정의**: 유명 브랜드/캐릭터 모방 또는 침해 의심 상품

**감지 키워드**:
```
- 나이키, 아디다스, 샤넬, 구찌, 루이비통, 애플, 삼성
- 아이폰, 갤럭시, nike, adidas, chanel, gucci, apple
- 디즈니, 마블, pokemon, 포켓몬, 짱구, 캐릭터
```

**판정**: **제외** (소싱 불가)
**이유**: 상표권/저작권 침해 시 형사처벌 (징역 7년 이하 또는 벌금 1억원 이하)

**[확인 필요]**: 의심 상품은 반드시 **KIPRIS**(특허정보검색서비스)에서 상표 등록 여부 확인

---

##### 2. **통관 리스크**
**정의**: 통관 제한 또는 특별 인증이 필요한 품목

**감지 키워드**:
```
- 의약품, 건강기능식품, 보조제, 영양제, 비타민
- supplement, vitamin, 프로바이오틱스, 오메가3
- 콜라겐, 단백질, 프로틴
```

**판정**: **보류** (벤더 확인 필요)
**이유**: 식약처 건강기능식품 인증 필요 (수입 신고서 + 성분 검사)

**벤더 질문** (자동 생성):
- 한국어: "한국 통관 시 필요한 서류나 인증이 있나요?"
- 中文: "韩国清关时需要什么文件或认证吗？"

---

##### 3. **의료기기 리스크**
**정의**: 의료적 효능 표방 또는 질병 치료 주장 상품

**감지 키워드**:
```
- 치료, 완치, 재생, 혈압, 혈당, 진단, 의료용
- medical, therapy, cure, 측정기, 모니터링, 검사
- 레이저, 초음파, 전기자극
```

**판정**: **제외** (소싱 불가)
**이유**: 의료기기법 위반 시 형사처벌 + 플랫폼 계정 영구 정지

**[확인 필요]**: 식약처 의료기기 품목 분류 확인 필수

### 2.2 리스크 판정 로직

```python
# sourcing_agent.py Line 292-309

def _make_decision(risk_flags, llm_analysis):
    # Rule 1: 치명적 리스크 → 제외
    if "지재권" in risk_flags:
        return "제외"

    # Rule 2: LLM 추천이 제외 → 제외
    if llm_analysis["recommended_decision"] == "제외":
        return "제외"

    # Rule 3: 의료기기 리스크 → 제외
    if "의료기기" in risk_flags:
        return "제외"

    # Rule 4: 기타 리스크 2개 이상 → 보류
    if len(risk_flags) >= 2:
        return "보류"

    # Rule 5: 안전 → 통과
    return "통과"
```

## 3️⃣ 2단계: AI 기반 상세 분석

### 3.1 Claude LLM 분석 항목

Fortimove는 **Claude 3.5 Sonnet** 모델을 사용하여 다음을 분석합니다:

1. **상품 분류** (3가지 카테고리)
   - **테스트**: 시장 반응 확인용 소량 소싱 (MOQ 10~50개)
   - **반복**: 검증된 Best-seller 지속 재고 (MOQ 100개+)
   - **PB**: Fortimove 브랜드로 리패키징 (MOQ 500개+)

2. **소싱 판정** (3가지 결과)
   - **통과**: 리스크 없음, 즉시 소싱 가능
   - **보류**: 벤더 추가 확인 필요
   - **제외**: 법적 리스크로 소싱 불가

3. **신뢰도 점수** (0.0 ~ 1.0)
   - 0.9 이상: 고신뢰 판정 (자동 진행)
   - 0.5 ~ 0.9: 중간 (수동 검토 권장)
   - 0.5 미만: 저신뢰 (필수 검토)

## 4️⃣ 3단계: 자동 승인 (Fortimove Golden Pass)

### 4.1 Golden Pass 기준

**🏆 5가지 조건을 모두 충족해야 자동 승인**

| 기준 | 요구사항 | 확인 방법 |
|------|----------|----------|
| **1. 소싱 판정** | 통과 | Sourcing Agent 결과 |
| **2. 마진율** | ≥ 45% | Pricing Agent 계산 |
| **3. 가격 범위** | ₩20,000 ~ ₩150,000 | 최종 판매가 |
| **4. KC 인증** | 불필요 | 키워드 감지 |
| **5. 리스크 플래그** | 0개 | Rule-based 필터 |

---

### 4.2 Golden Pass 로직

```python
# auto_approval.py Line 31-126

def evaluate(workflow_result):
    # 1. 소싱 판정 체크
    if sourcing_decision != '통과':
        return False, f"소싱 판정 불통과: {sourcing_decision}"

    # 2. 마진율 체크
    if margin_rate < 0.45:
        return False, f"마진율 부족: {margin_rate*100:.1f}% < 45%"

    # 3. 가격 범위 체크
    if not (20000 <= final_price <= 150000):
        return False, f"가격 범위 초과: ₩{final_price:,}"

    # 4. KC 인증 체크
    if kc_required:
        return False, "KC 인증 필요 상품"

    # 5. 리스크 플래그 체크
    if len(risk_flags) > 0:
        return False, f"리스크 플래그 {len(risk_flags)}개 감지"

    # 🏆 모든 조건 통과
    return True, "🏆 Fortimove Golden Pass - 모든 조건 충족"
```

## 5️⃣ 가격 책정 전략 (Pricing Agent)

### 5.1 Fortimove Premium Pricing 철학

**핵심 개념**: 단순 직구 대행이 아닌 **헬스케어 큐레이션 브랜드**

```
최종 판매가 = (원가 + 물류비 + 관부가세) × Curation Multiplier + Regulation Risk Cost
```

---

### 5.2 Curation Multiplier (카테고리별)

**정의**: 전문성 가중치 (1.3 ~ 1.5배)

| 카테고리 | Multiplier | 이유 |
|----------|-----------|------|
| **wellness** | 1.5x | 건강기능식품 - 최고 전문성 |
| **supplement** | 1.45x | 영양제 - 성분 검증 |
| **healthcare** | 1.5x | 헬스케어 기기 - 안전 보증 |
| **beauty** | 1.4x | 뷰티/화장품 - 피부 전문성 |
| **food** | 1.4x | 식품 - 위생 검수 |
| **fitness** | 1.35x | 운동용품 - 품질 보증 |
| **general** | 1.3x | 일반 상품 - 최소 마진 |

**비즈니스 근거**:
- 소비자는 "안전 보장"에 대해 30~50% 프리미엄 지불 의향
- 네이버 건기식 카테고리 평균 마진율: 40~60%
- Fortimove 타겟: 45% (업계 중상위)

---

### 5.3 Regulation Risk Cost (카테고리별 고정비)

**정의**: 법적 리스크 대응 비용 (인증, 보험, 법률 자문 등)

| 카테고리 | 고정비 | 포함 항목 |
|----------|--------|----------|
| **healthcare** | ₩20,000 | 의료기기 보험, 법률 자문 |
| **wellness** | ₩15,000 | 건기식 인증, 성분 검사 |
| **supplement** | ₩12,000 | 영양제 성분 검사 |
| **food** | ₩10,000 | 식품 위생 검사 |
| **beauty** | ₩8,000 | 화장품 안전성 검사 |
| **fitness** | ₩5,000 | 품질 보증 |
| **general** | ₩3,000 | 최소 보험 |

**비즈니스 근거**:
- 실제 Fortimove가 부담하는 인증/검사 비용 반영
- 판매가에 분산 포함 → 리스크 헤지

---

### 5.4 가격 계산 예시

**예시 1: Vitamin D3 영양제**

```
소싱 원가: ¥50 (CNY)
환율: 195원
무게: 0.2kg
카테고리: supplement

# 1. 기본 원가
원가 (KRW) = 50 × 195 = ₩9,750

# 2. 물류비
물류비 = 0.2kg × ₩30,000/kg = ₩6,000

# 3. 관부가세 (10%)
관부가세 = (9,750 + 6,000) × 0.1 = ₩1,575

# 4. 기본 원가 합계
기본 원가 = 9,750 + 6,000 + 1,575 = ₩17,325

# 5. Curation Multiplier 적용
큐레이션 원가 = 17,325 × 1.45 = ₩25,121

# 6. Regulation Risk Cost 추가
최종 판매가 = 25,121 + 12,000 = ₩37,121 → ₩37,900 (가격 정책)

# 7. 마진 계산
마진 금액 = 37,900 - 17,325 = ₩20,575
마진율 = 20,575 / 37,900 × 100 = 54.3%

✅ Golden Pass 통과 (마진율 54.3% ≥ 45%)
```

---

**예시 2: 일반 텀블러**

```
소싱 원가: ¥30 (CNY)
환율: 195원
무게: 0.5kg
카테고리: general

# 기본 원가
원가 = 30 × 195 = ₩5,850
물류비 = 0.5kg × ₩30,000/kg = ₩15,000
관부가세 = (5,850 + 15,000) × 0.1 = ₩2,085
기본 원가 합계 = ₩22,935

# 큐레이션 원가
큐레이션 원가 = 22,935 × 1.3 = ₩29,816

# 최종 판매가
최종 판매가 = 29,816 + 3,000 = ₩32,816 → ₩32,900

# 마진
마진 금액 = 32,900 - 22,935 = ₩9,965
마진율 = 30.3%

❌ Golden Pass 불통과 (마진율 30.3% < 45%)
→ 수동 검토 필요
```

## 1️⃣2️⃣ FAQ (자주 묻는 질문)

### Q1. 마진율 45%가 너무 높은 것 아닌가요?

**A**: 헬스케어 큐레이션 브랜드로서 **적정 마진**입니다.

- 네이버 건기식 카테고리 평균 마진율: 40~60%
- Fortimove는 "전문가 검수 + 안전 보장"이라는 부가가치 제공
- 인증/검사/법률 자문 비용을 감안하면 45%는 중간 수준

**대응 방안**:
- 통과율이 너무 낮으면 (< 10%) → 40%로 하향 조정 검토
- 통과율이 충분하면 (≥ 20%) → 현행 유지

---

### Q2. Golden Pass를 통과하지 못한 상품은 어떻게 하나요?

**A**: **수동 검토 절차**를 거칩니다.

1. PM이 직접 마진율/리스크 검토
2. 마진율 40~45%: 판매량 예측 후 승인 여부 결정
3. 마진율 < 40%: 소싱 포기
4. 리스크 플래그 1~2개: 벤더 확인 후 재판단

---

### Q3. 벤더가 KC 인증서를 제공하지 못하면?

**A**: **소싱 포기** 또는 **Fortimove 자체 인증 취득**

**옵션 1 (권장)**: 소싱 포기
- KC 인증 비용: 50만원~200만원/품목
- 인증 기간: 2~3개월
- ROI가 낮으면 포기

**옵션 2**: Fortimove 자체 인증 (PB 전환 시)
- MOQ 500개+ 대량 소싱 시만 고려
- 인증 비용을 판매가에 분산

---

### Q4. 건강기능식품 인증 없이 판매하면?

**A**: **형사처벌** 및 **플랫폼 영구 정지**

- 식약처 단속 시 형사고발 (징역 3년 이하 또는 벌금 3천만원 이하)
- 네이버/쿠팡 → 계정 영구 정지
- 소비자 피해 발생 시 → 손해배상 청구

**결론**: 절대 불가. 인증서 없으면 소싱 포기.

---

### Q5. 타오바오 상품명에 "Nike"가 있는데 정품일 수도 있지 않나요?

**A**: **정품 가능성 < 1%**. 즉시 제외 권장.

**이유**:
- 나이키 정품은 공식 유통망에서만 판매
- 타오바오의 "Nike"는 99% 이상 모조품
- 정품 확인 비용 > 소싱 포기 비용

**[확인 필요]**: 정품 의심 시 → KIPRIS에서 상표권자 확인 → 벤더에게 정품 인증서 요청
