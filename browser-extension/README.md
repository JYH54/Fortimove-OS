# FORTIMOVE Sourcing — Chrome 확장 프로그램

상품 페이지에서 **1클릭**으로 워크벤치에 전송하여 분석/콘텐츠 생성/상세페이지 리디자인을 자동화합니다.

## 지원 사이트
- 🇺🇸 **iHerb** (iherb.com, jp.iherb.com)
- 🇺🇸 **Amazon** (amazon.com, amazon.co.jp, amazon.co.uk)
- 🇨🇳 **타오바오** (taobao.com)
- 🇨🇳 **티몰** (tmall.com)
- 🇨🇳 **1688** (1688.com)
- 🇯🇵 **라쿠텐** (rakuten.co.jp)

## 설치 방법

### 1. 워크벤치 서버 실행 확인
```bash
curl http://localhost:8051/health
# → {"status":"healthy"} 응답이 와야 함
```

### 2. Chrome에 확장 프로그램 설치

1. Chrome 주소창에 `chrome://extensions` 입력
2. 우측 상단 **"개발자 모드"** 켜기
3. **"압축해제된 확장 프로그램을 로드합니다"** 클릭
4. `/home/fortymove/Fortimove-OS/browser-extension` 폴더 선택
5. 확장 프로그램 목록에 "FORTIMOVE Sourcing" 표시되면 완료

### 3. 사용법

1. **iHerb, Amazon, 타오바오** 등에서 상품 페이지 열기
2. Chrome 우측 상단 **F 아이콘** 클릭 (확장 프로그램 아이콘)
3. 자동으로 추출된 정보 확인:
   - 제목, 브랜드, 카테고리, 가격, 이미지 개수
4. **카테고리** 선택 + **원가/무게** 확인
5. **"워크벤치로 전송 & 분석 시작"** 클릭
6. 워크벤치 탭이 자동으로 열리며 Step 2 분석 화면으로 이동

## 주요 기능

### 자동 추출
- **JSON-LD 기반**: iHerb 등 구조화 데이터 우선 사용
- **DOM 폴백**: 플랫폼별 맞춤 셀렉터
- **OG 메타태그**: 폴백
- 최대 **10~15개 이미지** 자동 수집

### 자동 매핑
- 플랫폼 → 국가 자동 매핑 (iHerb JP → JP)
- 카테고리 → 워크벤치 카테고리 매핑
- 가격 → 숫자 자동 추출

### 워크벤치 연동
- 워크플로우 자동 실행 (빠른 검증 / 전체 등록)
- 이미지 자동 업로드 → Step 3 상세페이지 탭에 분류 표시
- Score/마진 자동 계산

## 차단 우회

브라우저에서 **대표님의 정상 세션**으로 크롤링되기 때문에:
- Amazon 서버 차단 우회 ✓
- iHerb 봇 탐지 우회 ✓
- 타오바오 IP 차단 걱정 없음 ✓

**주의:** 확장 프로그램은 대표님의 브라우저 세션을 사용합니다. 타오바오/Amazon 로그인 상태와 관련 없이 페이지 내용만 추출합니다.

## 개발자 정보

- Manifest Version: 3 (Chrome 88+)
- 필요 권한: activeTab, storage, scripting
- 워크벤치 서버: http://localhost:8051

## 문제 해결

### "상품 정보를 추출할 수 없습니다"
→ 페이지 새로고침 후 다시 시도

### "전송 실패: HTTP 500"
→ 워크벤치 서버 상태 확인 (`curl http://localhost:8051/health`)

### 아이콘에 빨간 점이 안 보임
→ 지원되지 않는 페이지입니다. 상품 상세 페이지에서만 작동합니다.
