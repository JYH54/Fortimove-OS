## 시스템 접속 방법

### 1️⃣ 웹 인터페이스 (추천)

**접속 URL**: [http://localhost:3000](http://localhost:3000)

브라우저에서 위 주소로 접속하면 **드래그 앤 드롭** 방식의 사용하기 쉬운 인터페이스를 사용할 수 있습니다.

### 2️⃣ API 문서 (고급 사용자)

**Swagger UI**: [http://localhost:8000/docs](http://localhost:8000/docs)

프로그래밍 방식으로 API를 호출하거나 자세한 기술 문서를 확인할 수 있습니다.

## 이미지 업로드 방법

### 방법 1: 드래그 앤 드롭 (가장 쉬움)

1. 타오바오/티몰 상품 상세페이지에서 이미지를 **우클릭 → 다른 이름으로 저장**
2. 웹 인터페이스([http://localhost:3000](http://localhost:3000))에 접속
3. 보라색 업로드 영역에 이미지 파일을 **드래그해서 놓기**
4. 최대 20개 이미지까지 한 번에 처리 가능

### 방법 2: 클릭하여 선택

1. 보라색 업로드 영역을 **클릭**
2. 파일 선택 창에서 이미지 선택 (Ctrl 또는 Shift 키로 여러 개 선택 가능)
3. "열기" 버튼 클릭

### 지원 파일 형식

- ✅ JPG/JPEG
- ✅ PNG
- ✅ WEBP

## 처리 옵션 설정

이미지를 업로드하면 **설정 패널**이 자동으로 나타납니다.

### 1. 📐 무드톤 선택

상품 컨셉에 맞는 이미지 스타일을 선택하세요:

| 무드톤 | 추천 상품 | 효과 |
|--------|-----------|------|
| **💎 프리미엄** | 명품, 고가 상품 | 채도 낮춤, 밝기 증가, 고급스러운 톤 |
| **💰 가성비** | 생활용품, 저가 상품 | 채도 증가, 선명한 색상, 눈에 띄는 비주얼 |
| **⚪ 미니멀** | 디자인 상품, 가전 | 채도 감소, 깔끔한 느낌 |
| **✨ 트렌디** | 패션, 악세서리 | 채도/명도 증가, 화사한 분위기 |

### 2. 🏷️ 브랜드 타입

- **Fortimove Global**: 테스트 상품, 단기 판매 상품 → 캐주얼한 번역
- **Fortimove**: 핵심 PB 상품 → 정제된 번역

### 3. ⚠️ 리스크 자동 처리

**체크 시** (권장):
- 유아 이미지 자동 삭제
- 사람 얼굴 블러 처리
- 브랜드 로고 제거

**체크 해제 시**:
- 리스크만 탐지하고 원본 유지 (수동 처리 필요)

## 주요 기능 설명

### 🔤 OCR (텍스트 추출)
- **엔진**: EasyOCR
- **지원 언어**: 중국어 간체, 영어
- **정확도**: 신뢰도 0.6 이상 텍스트만 추출

### 🤖 AI 번역
- **모델**: Claude 3.5 Sonnet
- **특징**:
  - 단순 번역이 아닌 **한국 이커머스 스타일 문구** 생성
  - 의료/의약품 오인 표현 자동 필터링
  - 유아용 제품 관련 금지 표현 제거

### 🛡️ 리스크 탐지
- **유아 이미지**: CLIP AI 모델 (정확도 70%)
- **얼굴 감지**: OpenCV Haar Cascade
- **브랜드 로고**: 주요 브랜드 DB 매칭 (Nike, Adidas, LV, Gucci 등)

### 🎨 이미지 재가공
- **해상도**: 1200px 너비로 자동 리사이징 (네이버/쿠팡 최적화)
- **품질**: JPEG 85% 압축 (용량 vs 품질 균형)
- **무드톤 조정**: HSV 색공간 변환으로 자연스러운 색감 조정

### 📝 SEO 자동 생성
- 상품명 3가지 스타일 자동 생성
- 검색 태그 10개 추출
- 핵심 키워드 5개 선정

## 문제 해결

### Q1. 웹 페이지가 열리지 않아요 (localhost:3000)

**해결 방법**:
```bash
# 컨테이너 상태 확인
cd /home/fortymove/Fortimove-OS/image-localization-system
docker-compose ps

# 모든 서비스가 "Up" 상태인지 확인
# frontend가 없다면 시작
docker-compose up -d frontend
```

### Q2. 이미지를 업로드했는데 처리가 안 돼요

**가능한 원인**:
1. **파일 형식 문제**: JPG, PNG, WEBP만 지원
2. **파일 크기 초과**: 개별 파일 20MB 이하
3. **개수 초과**: 최대 20개 이미지까지

**해결 방법**:
- 브라우저 개발자 도구(F12) → Console 탭에서 오류 확인
- 파일 크기 줄이기 또는 개수 제한 준수

### Q3. "API 오류" 메시지가 나타나요

**해결 방법**:
```bash
# 백엔드 로그 확인
docker logs image-localization-system-backend-1 --tail 50

# 백엔드 재시작
docker-compose restart backend
```

### Q4. 번역 품질이 이상해요

**확인 사항**:
1. **Anthropic API 키 유효성**: .env 파일 확인
2. **브랜드 타입 설정**: 상품에 맞는 타입 선택했는지 확인

### Q5. 처리 시간이 너무 오래 걸려요

**정상 처리 시간**:
- 1개 이미지: 10~20초
- 10개 이미지: 1~2분

**느릴 경우**:
- 첫 실행 시 AI 모델 다운로드로 시간 소요 (최초 1회)
- 이미지 해상도가 매우 높으면 처리 시간 증가

## 💡 사용 팁

### ✅ 최적의 이미지 선택
- 타오바오 상세페이지에서 **텍스트가 많은 이미지** 우선 업로드
- 제품 단독 컷보다 **설명이 포함된 이미지**가 SEO에 유리

### ✅ 무드톤 활용
- 동일 상품도 타겟층에 따라 무드톤 다르게 테스트 가능
- A/B 테스트용으로 여러 버전 생성 추천

### ✅ SEO 메타데이터 활용
- 상품명 3안 중 검색량 많은 키워드 포함된 것 선택
- 검색 태그를 네이버 쇼핑 검색어로 활용

### ✅ 리스크 관리
- 자동 처리 후에도 **최종 확인 필수**
- 특히 유아용품, 의료기기, 화장품은 수동 검수 권장
