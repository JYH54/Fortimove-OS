# 이미지 자동 현지화 워크플로우 (Image Processing)

**사용 목적**: 타오바오/티몰 상세페이지 이미지를 업로드하면 유아/인물 리스크 제거, 중국어→한국어 번역, 무드톤 조정, SEO 메타데이터까지 한 번에 처리합니다.

**기준 문서**: `docs/image-localization-sop.md` 및 `agents-spec.md`의 7번 에이전트 절차.

## 워크플로우 실행 순서

1.  **[이미지 현지화 에이전트]** 사용자가 업로드한 이미지에서 OCR로 중국어 텍스트를 추출하고, 유아/인물/브랜드 로고를 탐지합니다. (리스크 스캔)

2.  **[이미지 현지화 에이전트]** 탐지된 리스크 요소를 자동 처리합니다.
    *   유아 이미지: 삭제 또는 상품 단독 이미지로 교체
    *   인물 얼굴: AI 생성 인물로 교체 또는 블러 처리
    *   타사 로고: 영역 제거 (inpainting)

3.  **[이미지 현지화 에이전트]** AI를 활용하여 중국어 텍스트를 한국 이커머스 판매형 문장으로 번역합니다. (금지 키워드 필터링 적용)

4.  **[이미지 현지화 에이전트]** 사용자가 지정한 무드톤(프리미엄/가성비/미니멀)에 맞춰 색감을 조정하고, 번역된 한국어 텍스트를 이미지에 오버레이합니다.

5.  **[상품 등록 에이전트]** 재가공된 이미지를 분석하여 네이버/쿠팡 SEO 최적화 상품명 3안, 검색 태그 10개, 옵션명 정규화 목록을 자동 생성합니다.

6.  **최종 출력**:
    *   재가공 이미지 파일 (JPG/PNG)
    *   SEO 메타데이터 (JSON)
    *   리스크 처리 리포트

## 입력 요구사항

| 항목 | 필수 여부 | 형식 | 예시 |
|-----|---------|------|------|
| **이미지 파일** | 필수 | JPG/PNG (복수 가능) | product_detail_01.jpg |
| **무드톤 지정** | 필수 | 프리미엄/가성비/미니멀/트렌디 | "프리미엄" |
| **브랜드 타입** | 선택 | Fortimove Global / Fortimove | "Fortimove Global" |
| **원본 상품명** | 선택 | 텍스트 | "超强吸水速干毛巾" |

## 출력 결과물

```
processed_images/
├── product_detail_01_premium_kr.jpg  (재가공 이미지)
├── product_detail_02_premium_kr.jpg
└── metadata.json  (SEO 메타데이터)

metadata.json 구조:
{
  "seo": {
    "product_names": ["상품명 안1", "상품명 안2", "상품명 안3"],
    "tags": ["태그1", "태그2", ...],
    "options": [{"원본": "蓝色", "정규화": "블루"}],
    "keywords": ["키워드1", "키워드2", ...]
  },
  "risk_report": {
    "infant_detected": 2,
    "infant_removed": 2,
    "faces_detected": 3,
    "faces_replaced": 3,
    "logos_detected": 1,
    "logos_removed": 1
  }

```

## 실행 예시

**시나리오**: 타오바오 타월 상품의 상세페이지 이미지 5장을 "프리미엄" 무드톤으로 재가공

```
[입력]
- 이미지 5장 업로드
- 무드톤: "프리미엄"
- 브랜드: "Fortimove"

[처리 과정]
1. OCR 추출: "超强吸水" "宝宝专用" 등 중국어 텍스트 추출
2. 리스크 탐지: 유아 이미지 2장, 성인 얼굴 3명, 타사 로고 1개 탐지
3. 리스크 처리:
   - 유아 이미지 2장 삭제 → 상품 단독 이미지로 대체
   - 성인 얼굴 3명 → AI 생성 한국인 모델로 교체
   - 타사 로고 → inpainting으로 제거
4. 번역: "超强吸水" → "초강력 흡수력"
5. 무드톤: 채도 -30%, 명도 +20% (프리미엄 톤)
6. 텍스트 삽입: 나눔고딕으로 한국어 텍스트 오버레이
7. SEO 생성:
   - 상품명: "프리미엄 초강력 흡수 속건 마이크로화이버 타월"
   - 태그: 속건타월, 마이크로화이버, 프리미엄타월, ...

[출력]
- 5장의 재가공 이미지 (유아 이미지 제거됨)
- SEO 메타데이터 JSON
- 리스크 처리 리포트
```

## 주의사항

*   **유아 이미지 필수 제거**: 플랫폼 정책상 어린이제품법 리스크가 높으므로 무조건 삭제 또는 대체합니다.
*   **번역 품질 검수**: AI 번역 후 의료기기 오인 문구가 없는지 최종 확인합니다.
*   **무드톤 과도 조정 금지**: 원본 대비 ±30% 이내로 조정하여 허위 광고 소지를 방지합니다.
*   **무료 폰트만 사용**: 나눔고딕, 프리텐다드 등 저작권 없는 폰트만 사용합니다.
