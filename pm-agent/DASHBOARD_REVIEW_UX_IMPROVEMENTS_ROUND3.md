### 2. **정렬 기능 개선** ✅
**문제**: 점수가 낮은 순서로 정렬되어 있어 혼란스러움
**해결**:
- 기본 정렬: 점수 높은 순 (내림차순)
- 정렬 토글 버튼 추가: 점수 ↑↓, 날짜 ↑↓
- 버튼 클릭 시 오름차순/내림차순 전환
- 현재 활성 정렬 버튼 시각적 표시

**구현 위치**:
- JavaScript: [review_list.js:7-64](pm-agent/static/js/review_list.js#L7-L64)
- HTML: [review_list.html:40-46](pm-agent/templates/review_list.html#L40-L46)

**정렬 로직**:
```javascript
let currentSort = { field: 'score', order: 'desc' }; // 기본: 점수 높은 순

function toggleSort(field) {
    if (currentSort.field === field) {
        // 같은 필드면 오름차순/내림차순 토글
        currentSort.order = currentSort.order === 'desc' ? 'asc' : 'desc';
    } else {
        // 다른 필드면 해당 필드 내림차순으로 변경
        currentSort.field = field;
        currentSort.order = 'desc';
    }
    renderReviewList();
}
```

### 3. **Review Detail 페이지 대대적인 재구성** ✅
**문제**: "Review 채널 들어가서는 대체 어떻게 일을 하라는지 확인 자체가 안된다"
**해결**: 카드 기반 레이아웃으로 정보 계층 구조화

#### 3-1. 왼쪽 컬럼 (읽기 전용 - AI 생성 콘텐츠)
**구조 변경**:
```
이전: 1개의 큰 카드에 모든 정보
↓
현재: 4개의 구분된 카드
  1. 📦 소싱 정보 (파란색)
  2. 🤖 AI 분석 요약 (청록색)
  3. 🟢 네이버 스마트스토어 (녹색 - AI 생성)
  4. 🟠 쿠팡 로켓배송 (주황색 - AI 생성)
```

**주요 개선점**:
- 각 채널별 콘텐츠를 독립된 카드로 분리
- 색상 코딩으로 채널 구분 명확화
- "참고용" 배지 추가로 읽기 전용임을 명시
- white-space: pre-wrap으로 설명 텍스트 가독성 개선

#### 3-2. 오른쪽 컬럼 (편집 가능 - 검수 콘텐츠)
**구조 변경**:
```
이전: 1개의 카드에 네이버/쿠팡/가격 혼재
↓
현재: 3개의 구분된 카드
  1. ✏️ 네이버 스마트스토어 (검수 & 편집) - 녹색 테두리
  2. ✏️ 쿠팡 로켓배송 (검수 & 편집) - 주황색 테두리
  3. 💰 가격 및 검수 노트 - 청록색 테두리
```

**주요 개선점**:
- 각 채널별 편집 영역을 독립된 카드로 분리
- "← 왼쪽 AI 내용 참고" 배지로 참조 방향 안내
- 필수 필드 `<span class="text-danger">*</span>` 표시
- 각 입력 필드마다 상세한 가이드 텍스트 (최대 글자수, 권장사항)
- 두꺼운 테두리 (`border-width: 2px`)로 편집 영역 강조

**입력 필드 가이드 예시**:
```html
<small class="text-muted">네이버 스마트스토어 등록용 제목 (최대 50자 권장)</small>
<small class="text-muted">마진율을 고려한 최종 판매 가격 (왼쪽 AI 추천가 참고)</small>
<small class="text-muted">운영팀 내부 메모 (고객에게 노출되지 않음)</small>
```

### 4. **워크플로우 안내 개선** ✅

## 🎨 디자인 시스템

### 색상 코딩
- **파란색 (primary)**: 소싱 정보
- **청록색 (info)**: AI 분석 요약, 가격/노트
- **녹색 (success)**: 네이버 채널
- **주황색 (warning)**: 쿠팡 채널
- **회색 (secondary)**: 워크플로우 안내

### 카드 계층 구조
```
카드 헤더 색상 = 카테고리 식별
├─ 배지 (참고용/편집 가능)
└─ 카드 본문
   ├─ 레이블 (fw-bold small text-muted)
   ├─ 입력 필드 또는 표시 영역
   └─ 가이드 텍스트 (text-muted)
```

### 시각적 계층
1. **최상위**: 네비게이션 바 (bg-dark)
2. **콘텐츠 레벨1**: 좌우 2컬럼 (col-md-5, col-md-7)
3. **콘텐츠 레벨2**: 카드 (색상 구분)
4. **콘텐츠 레벨3**: 카드 내 섹션 (mb-2, mb-3)
